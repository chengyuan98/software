以上软件都自带数字签名，可右键点击程序，查看“属性”--“数字签名”--“详细信息”，

弹出窗口会显示“数字签名正常”，以及软件厂商名称等信息，可供查询。

这里的翻墙软件均源自官方网站，并经过数字签名及杀毒软件检测，请放心使用。
